#Floral 是一个很生活化的 APP,具有专题、社区、商城、我的 四个大模块	
====
#个人很喜欢这种小清新的 APP;因为我在家自学 iOS快5个月了， 所以就拿这款 APP 练练手
====
###这个项目花了我一周时间:9.25~10.1 
====
####暂时完成了专题大部分功能：	
- 1.布局切换
- 2.collectionCell 内部控件的自动布局
- 3.点击 cell 后加载 html 页面
- 4.搜索
- 5.分类页面
- 6.分类页面的点击跳转等	
####社区的框架搭建:
- 1.广告无限轮播
- 2.标签 cell 
- 3.TOP10
####我的页面搭建:
- 1.xib 自动布局子控件
- 2.给后面的是否已经登录判断预留接口
